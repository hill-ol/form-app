# Animation improvement plans

Generated by the `improve-animations` skill audit on 2026-07-22, stamped at commit `fa3cf33`. All 11 plans were written and then implemented directly against the working tree in the same session (not isolated worktrees).

Note: plans 001 and 003 originally named the shared tokens `--ease-out`/`--ease-in-out`/`--ease-drawer`. During implementation this collided with Tailwind v4's own reserved `--ease-out`/`--ease-in-out` theme tokens (defined in `tailwindcss/theme.css` as different cubic-beziers), silently overriding nothing and leaving the custom curves inert. All tokens and references were renamed to `--motion-ease-out`/`--motion-ease-in-out`/`--motion-ease-drawer` to avoid the collision — see `src/app/globals.css`.
